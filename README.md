# MNIST with TensorFlow

## Getting started

* Create a virtualenv
    * `virtualenv .venv`
* Activate it
    * `source .venv/bin/activate`
* Install python dependencies
    * `pip install -r requirements.txt`
* Rune script files like this
    * `python NN_simple_scratch.py`
    
## Tensorboard

* Open tensorboard
    * `tensorboard --logdir=./tmp/train`
